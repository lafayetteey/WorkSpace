폼 관련 태그

form 관련태그는 실제 브라우저 사용자가 입력할 수 있는 양식을 제공하는태그 (ex: 로그인 기능) form 태그 내의 input 이라는 태그들을 통해 사용자가 입력한 정보를 서버로 넘기는 역할을 한다(submit) * action : 폼 의 입력된 값들을 전송받을 서버의 이름
* method : get 또는 post 방식으로 전송방식을 지정 get : 사용자가 입력한 정보를 그대로 주소창에 띄움(검색값등에 쓰임) post : 민감한 정보(id 또는 pw) 를 주소창에 띄우지 않음

 

submit : 버튼을 눌렀을때 action 에 지정한 경로로 method 에 지정한 방식으로 input 태그에 입력한 값을 전달한다.
